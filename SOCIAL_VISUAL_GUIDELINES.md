# 仙加味社群圖片正式規則

更新日期：2026-07-21

## 1. 編號規則

- 圖面不得顯示流水號、篇數、Day、No.、01、02、21、30 等系列編號。
- 貼文標題不得顯示內部排序編號。
- 產品必要規格數字必須保留，例如 30cc、75g、100g、180cc、600g、8 塊、32 塊。
- 檔名與資料庫 ID 可保留內部編號，但不可出現在對外圖面與貼文標題。

## 2. 產品硬規則

- 只能使用仙加味已核准的產品原圖、DM 與正式素材。
- 不重畫產品，不改包裝、盒型、瓶型、袋型、標籤、比例或高矮胖瘦。
- 只允許去背、等比例縮放、接地陰影與環境光融合。
- 龜鹿飲 30cc 必須維持矮胖玻璃小瓶；180cc 必須維持正式鋁袋外觀。

## 3. 小老闆角色固定設定

- Q 版小男孩、短黑髮、大眼睛、微紅臉頰、親切笑容。
- 米白色中式上衣、深橄欖綠圍裙。
- 圍裙胸前為紅色直式「仙加味」印章 Logo。
- 小鹿與小烏龜為兩個分開圖案，不得合體。
- 人物可調整動作與視線，但不得改變角色五官、服裝與品牌識別。

## 4. 動作姿勢輪替

每批圖片應交錯使用下列情境，不可全部站立看鏡頭：

1. 雙手拿產品介紹。
2. 單手指向標題或資訊卡。
3. 在桌邊示範沖泡、攪拌或取用。
4. 坐在木桌旁閱讀包裝標示。
5. 從藥櫃取出原料或整理抽屜。
6. 端著熱飲、湯碗或保溫壺。
7. 與小鹿、小烏龜分別互動。
8. 側身介紹、回頭招呼、微彎身展示等自然動作。

同一批中相鄰兩張不得使用完全相同的姿勢；同一姿勢最多使用兩次，且背景與構圖必須不同。

## 5. 版面配置輪替

至少輪替以下構圖：

- 左人物、右文字。
- 右人物、左文字。
- 上方主標、下方情境與產品。
- 中央產品主視覺、人物從側邊引導。
- 木桌／藥櫃／窗邊等情境式構圖。
- 大留白品牌語錄構圖。
- 步驟式資訊卡，但不使用篇數或流水號。

不得把所有貼文套在同一個深藍標題框、同一個米色文字區與同一個人物位置。

## 6. 品牌視覺

- 背景：溫暖米色、藥房木質、柔和自然光。
- 主色：深藍 `#0B1F3B`、深綠、紅印章。
- 點綴：少量金色與橘色，不可搶過產品。
- 字體：繁體中文，標題清楚、內文精簡，手機縮圖仍可閱讀。
- 保持高級留白感，避免滿版堆字、過多框線或廉價促銷感。

## 7. 圖片輸出與驗收

- 尺寸：1254 × 1254 px。
- 格式：清晰 PNG；不得模糊、截圖放大或二次壓縮失真。
- 文字不得壓到人物臉部、產品標籤或 Logo。
- 同批圖片需同一品牌系統，但人物姿勢、背景與構圖要有變化。
- 新圖片只建立待審貼文，通過人工確認後才可排程或發布。
