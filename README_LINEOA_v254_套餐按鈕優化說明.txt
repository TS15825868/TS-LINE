仙加味 LINE OA v254｜套餐按鈕優化版

本版修改重點：

1. 主快捷選單新增「看套餐優惠」
   客人不用先進價格方案，也能直接看到套餐輪播卡。

2. 價格方案內「套餐搭配」改成「套餐優惠」
   名稱更直覺，客人比較知道這是優惠組合。

3. 套餐卡片按鈕改成：
   - 加入購物車
   - 直接下單
   - 看其他套餐
   - 詢問客服

4. 客人輸入以下文字會進套餐：
   - 套餐
   - 搭配
   - 搭配組合
   - 套餐優惠
   - 看套餐優惠

5. 客人輸入「優惠、方案、折扣、買多有優惠」會先進價格方案選單，
   再讓客人選單品優惠、活動優惠或套餐優惠。

保留功能：
- 產品DM單張圖
- 購物車
- 階梯優惠
- 結帳
- CRM
- 健康敏感問題轉人工方向

上線方式：
把此包覆蓋 Render 專案後 redeploy。
