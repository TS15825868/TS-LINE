# 仙加味 LINE OA v287｜傳統食補用途方向同步版

## 本版主軸
- 讓客人先認識龜鹿食補與各產品用途方向。
- 以固定取用、方便即飲、沖泡燉湯、傳統大規格、自行搭配五個方向協助選擇。
- 價格、活動與下單流程維持 LINE OA 處理。
- 《本草綱目》與藥典保留名稱、食補文化與品質規格用途，不直接替產品做療效。

## 主要檔案
- server.js：LINE Bot 主程式。
- data.json：產品、價格、用途方向、FAQ、套餐與品牌資料。
- Code.gs：沿用 v231_stable，未修改。
- LINEOA_正式回覆文案_v287.md：客服與測試參考。
