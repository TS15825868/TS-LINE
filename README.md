# 仙加味 LINE Bot v66 完整成交優化版

## 必填環境變數
- `CHANNEL_ACCESS_TOKEN`
- `CHANNEL_SECRET`
- `CRM_URL`（可選，用於 Google Sheet 訂單記錄）

## Rich Menu 六格訊息
上排：
1. 看產品 → `看產品`
2. 查看購買清單 → `查看購買清單`
3. 幫我推薦 → `幫我推薦`

下排：
4. 搭配組合 → `看搭配組合`
5. 怎麼使用 → `怎麼使用`
6. 直接下單 → `我想直接下單`

## v66 重點
- 商品 / 套餐卡固定兩個成交入口：加入清單、直接買。
- 結帳中的付款與配送維持按鈕化。
- 查看購買清單有繼續加商品、移除商品、清空清單、直接結帳。
- 清空 / 刪除命令最高優先，不會被商品關鍵字誤吃。
- 第一則網站 deep link 訊息不會被歡迎卡吃掉。
- 敏感問題縮窄，不會把一般「適不適合」導流誤轉中醫師。
- 訂單送出前有確認步驟，降低錯單。
