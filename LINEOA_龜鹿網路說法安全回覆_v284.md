# 仙加味 LINE OA｜龜鹿網路說法安全回覆 v284

## 可自動接住的問題
- 龜鹿產品有什麼功效／好處？
- 可以改善疲勞、貧血、掉髮、腰痠、耳鳴、性功能或骨質疏鬆嗎？
- 空腹吃吸收最好嗎？
- 沒有骨質疏鬆需要吃來預防嗎？
- 懷孕、哺乳、小孩、服藥或慢性病可以使用嗎？
- 可以取代藥物或醫療嗎？

## 回覆原則
1. 不宣稱治療、改善、預防疾病或症狀。
2. 不使用「空腹吸收最佳」作為所有人的固定說法。
3. 只回答產品型態、規格、成分、使用方式與保存方式。
4. 疾病、症狀、孕哺、兒童、用藥與特殊體質導向醫師、中醫師或藥師。
5. 不可說產品能取代藥物、骨密度檢查、診斷、治療或醫療追蹤。
