仙加味 LINE OA v231｜server.js 修正版

本包已直接把新版 server.js 覆蓋到專案內。

本版修正：
1. 官網點 LINE 帶入「我要詢問【龜鹿飲 30cc】」會直接接到產品回覆。
2. 可辨識：
   - 我要詢問【龜鹿飲 30cc】
   - 我要詢問【龜鹿飲30cc】
   - 龜鹿飲 30cc
   - 龜鹿飲30cc
   - 龜鹿飲180cc
   - 龜鹿膏
   - 龜鹿湯塊
   - 龜鹿膠
   - 鹿茸粉
3. 「看產品DM」會回 DM 頁連結，不會跳一般選擇題。
4. 移除重複 function，避免後面覆蓋前面。
5. 版本文字統一 v231。
6. 保留購物車、階梯優惠、結帳、CRM 寫入、健康檢查 /healthz。

部署方式：
- 將本 ZIP 上傳到 GitHub / Render 對應 LINE OA 專案
- 或直接用裡面的 server.js 覆蓋原本 server.js
- Render 重新部署後，到 /healthz 檢查 version 是否為 v231

注意：
- 如果 Render 免費方案休眠，第一次回覆仍可能慢 5～30 秒。
- 這不是程式錯誤，是冷啟動造成。
