## LINE OA Manager 設定

1. 選擇大型 3×2 版型。
2. 上傳 `xianjiawei-rich-menu-2500x1686-v307.jpg`。
3. 六個區域均選擇「文字」動作。
4. 依上表輸入完全相同的傳送文字。
5. 顯示期間可設為長期；選單列文字建議使用「仙加味服務」。
6. 儲存並發布後，逐一測試六個按鈕。
