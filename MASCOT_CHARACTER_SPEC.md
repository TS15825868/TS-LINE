# 仙加味小老闆｜LINE OA 固定角色規範

版本：2026-07-10 Final

## 固定角色設定

「仙加味小老闆」是一位以小男孩為原型的 Q 版品牌角色：短黑髮、大眼睛、臉頰微紅、笑容親切，整體可愛、溫暖、有禮貌。

## 固定服裝設定

- 米白色中式上衣
- 深橄欖綠圍裙
- 圍裙胸前使用紅色直式「仙加味」印章 Logo
- 圍裙上保留彼此分開的小鹿與小烏龜圖案
- 小鹿與小烏龜不可合體、不可融合

## 固定品牌風格

溫暖米色、深藍、深綠、印章紅、金色點綴；可愛 Q 版但不幼稚，保有仙加味中藥房、老字號、品牌感與溫暖親切感。

## LINE OA 使用規則

- 歡迎卡、推薦卡、搭配導覽卡與品牌故事卡均沿用同一角色
- LINE Flex Message 角色圖使用官網正式素材：`https://ts15825868.github.io/xianjiawei/images/brand/xianjiawei-mascot.jpg`
- 人物臉型、髮型、比例、服裝、圍裙、紅印章 Logo、小鹿與小烏龜不得更換
- 只可更換動作、表情、使用情境與文字
- 所有產品規格、價格、使用方式與按鈕功能仍以 `data.json` 與 `server.js` 正式資料為準

## 現行程式位置

- 角色圖路徑：`server.js` 中的 `MASCOT_PATH`
- 歡迎卡：`mascotWelcomeReply()`
- 角色卡共用樣式：`mascotBubble()`
- 推薦與搭配導覽：`recommendReply()`、`comboMenuReply()`

此規範為仙加味小老闆最終定稿，後續不得自行更換角色或服裝。
